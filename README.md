# Система для ведення обліку аналізів води




## Стек технологій
1. PHP 8.3
2. Laravel v10
3. MySql 
4. VueJS 3
## Запуск проекту

Щоб запустити додаток, потрібно встановити залежності

```bash
  composer install
```

```bash
  npm install
```

Для запуску бекенд частини потрібно виконати команду
```bash
    php artisan serve
```

Для запуску фронтенд частини команда
```bash
    npm run dev
```
